# homebrew-claude-codex-usage

Homebrew tap for Claude Codex Usage.

## Install

```bash
brew tap ShipItAndPray/claude-codex-usage
brew install --cask claude-codex-usage
```

Optional variants:

```bash
brew install --cask claude-codex-usage-claude
brew install --cask claude-codex-usage-codex
```

## Casks

- `claude-codex-usage`: combined Claude + Codex app
- `claude-codex-usage-claude`: Claude-only app
- `claude-codex-usage-codex`: Codex-only app
